01 核心体验

写作工具应该让内容成为主角

从打开文件到完成发布，界面保持克制，能力保持完整。

01

所见即所得

直接编辑渲染后的 Markdown 内容，不在源码和预览之间来回切换。

02

本地文件工作区

打开文件夹、搜索文档、管理标签页，文件始终保存在你自己的磁盘上。

03

Mermaid 流程图

平时展示清晰图表，需要修改时再进入源码，保持阅读与编辑的平衡。

04

主题与字体

提供经典明暗主题，并可选择系统中的文本、代码和界面字体。

05

结构化大纲

自动识别标题层级，快速定位长文中的具体段落，并可随时收起。

06

多种导出方式

从公众号样式 HTML 到可部署的网站项目，导出能力通过插件持续扩展。

02 开放扩展

编辑器保持简单，能力交给插件

命令、面板、Markdown 块和导出器都通过公开 API 注册。插件权限默认拒绝，由宿主明确授权。

插件开发指南 查看示例
PLUGIN.TSPUBLIC API / 0.1
$ export default {
  manifest: {
    id: "hello.tickpad",
    permissions: ["document:read"]
  },
  activate(ctx) {
    ctx.commands.register({
      id: "hello.say",
      title: "Say hello",
      run: () => console.log("hello")
    })
  }
}
