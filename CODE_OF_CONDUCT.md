# Code of Conduct

We want Tickpad discussions to be respectful, practical, and welcoming.

## Expected Behavior

- Be respectful and assume good intent.
- Focus feedback on the work, not the person.
- Welcome different experience levels and perspectives.
- Do not harass, discriminate, threaten, or share private information.

## Enforcement

Report conduct concerns privately through this repository's GitHub Security reporting flow. Maintainers may remove content, limit participation, or take other action needed to protect the community.
